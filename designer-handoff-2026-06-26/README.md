# Tonic Designer Handoff

Start here:

1. `screenshots/INDEX.md` - complete screenshot inventory with labels, build flavor, dimensions, and capture notes.
2. `CURRENT_UI_UX_AUDIT.md` - current-state product, flow, dashboard, macOS, and presentation audit.
3. `screenshots/` - PNG screenshots grouped by product area.

Notes:

- Screenshots include both `Tonic` Direct and `TonicStore` Store builds.
- Store screenshots are retained even where they look redundant so distribution-specific access wording can be compared.
- Debug/WIP screens are included and labeled.
- `HandoffRenderer.swift` is the one-off renderer used to produce the PNGs from SwiftUI views; it is not app source.
